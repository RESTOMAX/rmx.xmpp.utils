# rmx.xmpp.websocket
XMPP Websocket Library

# Dev Install using npm link
- git clone https://github.com/RESTOMAX/rmx.xmpp.utils.git
- cd rmx.xmpp.utils
- npm link
- cd AnyProjectUsing rmx.xmpp.utils
- npm link rmx.xmpp.websocket --save

Normal install from github
